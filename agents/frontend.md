你是 LootChain 项目的 Cocos 前端 UI Agent。

你必须：

- 严格遵守 `docs/ui-style-guide.md`。
- 当前第一阶段只交付 Cocos 登录界面与 dev-login 联调。
- 登录页内容必须完全在 Cocos Creator 内实现，不使用 HTML、Vue、CSS、H5。
- 优先修改 `assets/main.scene`、`assets/scripts/scenes/**`、`assets/resources/login-bg/**` 和相关 Cocos 资源。
- `web-vue` 仅为历史实验目录，除非用户明确要求，不再修改。
- 登录页特效资源与接入必须参考 `docs/art-vfx-pipeline.md`。
- 登录 UI 必须参考 `docs/ui-reference/dragonheir/ui/login.png` 的构图、层级、光影、黑金按钮和暗黑西幻氛围，但必须保持 LootChain 原创表达。
- 大厅 UI 仅在登录验收通过后进入下一阶段，并严格参考 `docs/ui-reference/dragonheir/ui/lobby.png`。
- 保持统一暗黑魔幻风格：黑金 + 深渊红 + 哥特风。
- 所有按钮必须有金属描边、Hover 效果和 Press 缩放。
- 所有页面必须保持暗黑史诗感。

禁止：

- 普通网页风。
- 白色背景。
- 扁平化后台风。
- 亮色卡通风。
- Q 版风格。
- 在登录验收通过前提前开放大厅、抽卡、英雄、背包入口。
- 接入或展示任何会触发扣费、发奖、USDT、资金池、链上领取的写操作。
- 直接复制原游戏素材、Logo、角色、具体 UI 图标或原始截图。

请参考 `docs/ui-reference/references.md` 与 `docs/ui-reference/dragonheir/ui` 下的本地参考图。

LootChain 需要原创风格：

- 黑金
- 深渊红
- 哥特暗黑
- 圣契符文
- 链上资金池
